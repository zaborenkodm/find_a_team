2. Пользователи 
 
2.1 Просмотр рейтинга пользователей
Я, как залогинившийся пользователь,

а) захожу на главную страницу​ сайта.

б) Вижу там 10 пользователей с наивысшим рейтингом.
Рейтинг пользователей - составной параметр, определяется 
количеством посещенных и созданных мероприятий

в) У каждого игрока вижу:
- количество посещенным мероприятий 
- количество созданных  мероприятий

г) Вижу список 10 последних созданных мероприятий.

д) У каждого мероприятия вижу:
- название мероприятия
- адрес проведения
- дата, время
 
е) Вижу кнопки:
- Профиль
- Создать мероприятие
- Поиск мероприятия
- Поиск на карте

2.2 Просмотр профиля пользователя 
Я, как залогинившийся пользователь,

а) захожу на главную страницу​ сайта
и нажимаю кнопку "Профиль".

б) В результате попадаю
на страницу профиля

и вижу следующие поля:
- имя
- Аватар
- Контакт (e-mail)
- Список интересов
- Список созданных мероприятий
- Список посещенных мероприятий

У мероприятия вижу следующие поля:
- название мероприятия
- адрес проведения
- дата, время

2.3 Редактирование профиля
Я, как залогинившийся пользователь,

а) захожу на главную страницу​ сайта
и нажимаю кнопку "Профиль".

б) В результате попадаю
на страницу профиля

в) Нажимаю кнопку "Редактировать профиль".
В результате для редактирования становятся доступны
следующие поля:
- имя
- Аватар
- Контакт (e-mail)
- Список интересов

г) После внесения исправлений,
нажимаю кнопку "Сохранить".

В результате вижу страницу своего профиля
с отредактированными данными.

2.4 Удаление профиля
Я, как залогинившийся пользователь,

а) захожу на главную страницу​ сайта
и нажимаю кнопку "Профиль".

б) В результате попадаю
на страницу профиля

в) Нажимаю кнопку "Удалить профиль".
В результате мой профиль удаляется из системы.


2.5 Вход на сайт зарегистрированного пользователя
Я, как зарегистрированный пользователь,

а) захожу на главную страницу сайта
и нажимаю кнопку "Вход".

б) В результате попадаю
на страницу ввода логина и пароля.

Заполняю эти поля и нажимаю кнопку "Применить".

в) В результате вижу на главной странице сайта
вместо кнопок "Вход" и "Регистрация",
свой логин и кнопку "Выход".


2.6 Регистрация нового пользователя
Я захожу на главную станицу сайта
а) нажимаю кнопку "Регистрация".

б) В результате попадаю на страницу
редактирования профиля,
где вижу следующие поля для заполнения:
- имя
- Аватар
- Контакт (e-mail)
- Список интересов

в) После заполнения полей
нажимаю кнопку "Сохранить".

г) В результате вижу свой сохраненный профиль.
И становятся активны кнопки 
"Редактировать профиль" и "Удалить профиль".


3. Мероприятия

3.1 Создание мероприятия
Я, как залогинившийся пользователь,

а) захожу на главную страницу сайта, 
нажимаю кнопку "Создать мероприятие" 

б) в результате попадаю на страницу создания мероприятия.
Заполняю следующие поля:
- Название
- Адрес
- Дата, время
- указываю платное или бесплатное
- указываю цену, если платное
- при необходимости загружаю фотографию
- можно заполнить поле дополнительной информации

в) После заполнения всех необходимых полей

нажимаю кнопку "Сохранить мероприятие".

3.2 Редактирование мероприятия

Я, как залогинившийся пользователь,
создавший мероприятие,

а) захожу на страницу этого мероприятия,
нажимаю кнопку "Редактировать мероприятие",

в результате:

переадресован на страницу редактирования мероприятия,

б) изменяю любые из таких-то полей:
- Название
- Адрес
- Дата, время
- Платное/бесплатное
- Цена (если платное)
- Фотография
- Дополнительная информация

нажимаю "Сохранить",

в результате:

переадресован на страницу просмотра мероприятия,
вижу изменения в соответствующей изменённым полям информации

так что:

имею возможность исправить ошибку, обновить информацию о мероприятии, чтобы 
донести до заинтересованных людей актуальные сведения

3.3 Присоединение к мероприятию
Я, как залогинившийся пользователь,

а) захожу на главную страницу, 
и щелкаю по блоку интересующего мероприятия.

б) В результате:
переадресован на страницу просмотра мероприятия

в) У нужного мероприятия
нажимаю кнопку "Присоединиться".

г) В результате
я вижу себя в списке подписавшихся на мероприятие.

3.4 Отменить участие в мероприятии
Я, как залогинившийся пользователь,

а) захожу на главную страницу, 
и щелкаю по блоку интересующего мероприятия.

б) В результате:
переадресован на страницу просмотра мероприятия

в) У нужного мероприятия
нажимаю кнопку "Отменить участие".

г) В результате
я пропадаю из списка подписавшихся на мероприятие.


4. Комментарий

4.1 Создание комментария
Я, как залогинившийся пользователь,

а) захожу на страницу мероприятия 
и нажимаю кнопку "Создать комментарий" под мероприятием.

б) После заполнения поля комментария нажимаю кнопку "Сохранить". 
В результате я увижу свой комментарий под событием.

в) У каждого комментария выводятся поля:
- комментарий,
- автор (добавляется автоматически),
- время создания или время последнего редактирования (добавляется 
автоматически).

4.2 Редактирование комментария
Я, как залогинившийся пользователь,
создавший комментарий,

а) захожу на страницу мероприятия,
у которого хочу отредактировать свой комментарий,

б) напротив своего комментария 

нажимаю кнопку "Изменить",

в результате попадаю в окно редактирования комментария

в) после внесения изменений 

нажимаю кнопку "Сохранить",

в результате попадаю на страницу мероприятия 
и вижу свой отредактированный комментарий.

4.3 Удаление комментария
Я, как залогинившийся пользователь,
создавший комментарий,

а) захожу на страницу мероприятия,
у которого хочу удалить свой комментарий,

б) напротив своего комментария нажимаю кнопку "Удалить",
в результате мой комментарий пропадает


5. Поиск мероприятий по типу, дате, месту проведения

5.1 Поиск мероприятий
Я, как залогинившийся пользователь,

а) захожу на главную страницу сайта 

и нажимаю кнопку "Поиск мероприятия".

б) В результате попадаю в окно поиска,
в котором задаю поиск мероприятий по параметрам.

Для этого заполняю поля : 
- тип мероприятия,
- интервал дат,
- место проведения (округ или регион).

в) После заполнения интересующих полей 

нажимаю кнопку "Применить".

г) В результате выводится список мероприятий,
из которых можно выбрать,
к какому присоединиться.


5.2 Поиск мероприятия по привязке к карте

Я, как залогинившийся пользователь,

а) захожу на главную страницу сайта
и нажимаю кнопку "Поиск на карте".

б) В результате попадаю в окно поиска,
выбираю мышкой на карте города районы или определенные улицы.
По умолчанию выводится карта города, указанного в профиле пользователя,
но можно выбрать любой город из списка городов.
Также могу выбрать район или улицу из спика, 
который находится справа от карты города.

После выбора нажимаю кнопку "Применить".

в) В результате выводится список мероприятий,
из которых можно выбрать,
к какому присоединиться.
